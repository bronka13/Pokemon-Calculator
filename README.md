# Pokémon Calculator

Pokémon Calculator is an AHK v2 Poké Ball throwing simulator and calculator.

It uses [PokeAPI](https://pokeapi.co/) data, extracted with `./Python/import_pkmn_data.py` to `./AHK v2/PkmnDataAPI.ahk`, which is `#Included` in `./AHK v2/Datasets.ahk`.

## Usage

Run `./AHK v2/Pokéball Calculator.ahk` and use the GUI to input desired values. Whatever values aren't relevant to your calculation might be left untouched.
